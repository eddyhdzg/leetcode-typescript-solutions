# 1. Two Sum

### O(n) - TS with hash map

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

Runtime: 1 ms · Beats 86.85%
Memory: 56.29 MB · Beats 79.60%

## Code

```typescript
function twoSum(nums: number[], target: number): number[] {
  const hash: Record<number, number> = {};

  for (let i = 0; i < nums.length; i++) {
    const temp = target - nums[i];
    const pair = hash[temp];

    if (pair !== undefined) return [pair, i];
    hash[nums[i]] = i;
  }

  return [];
}
```
