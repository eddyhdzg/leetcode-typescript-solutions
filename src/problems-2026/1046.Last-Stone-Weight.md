# 1046. Last Stone Weight

### O(n log n) - TS with Heap

## Complexity

- Time: $$O(n \log n)$$
- Space: $$O(n)$$

## Submission

Runtime: 5 ms · Beats 37.94%
Memory: 55.88 MB · Beats 77.47%

## Code

```typescript
import { MaxHeap } from "@datastructures-js/heap";

function lastStoneWeight(stones: number[]): number {
  const heap = new MaxHeap<number>();
  for (const stone of stones) heap.push(stone);

  while (heap.size() > 1) {
    const y = heap.pop()!;
    const x = heap.pop()!;
    if (y !== x) heap.push(y - x);
  }

  return heap.size() === 1 ? heap.top()! : 0;
}
```
