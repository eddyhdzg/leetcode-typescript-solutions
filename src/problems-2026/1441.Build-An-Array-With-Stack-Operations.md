# 1441. Build An Array With Stack Operations

### O(n) - TS with Set()

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

Runtime: 0 ms · Beats 100.00%
Memory: 55.26 MB · Beats 66.49%

## Code

```typescript
function buildArray(target: number[], n: number): string[] {
  const stack: ("Push" | "Pop")[] = [];
  const set = new Set(target);

  for (let i = 1; i <= n && set.size > 0; i++) {
    if (set.has(i)) {
      stack.push("Push");
      set.delete(i);
    } else {
      stack.push("Push", "Pop");
    }
  }

  return stack;
}
```
