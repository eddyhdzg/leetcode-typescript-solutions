# 1470. Shuffle The Array

### O(n) - TS with two pointers

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

Runtime: 59 ms · Beats 20.94%
Memory: 58.98 MB · Beats 31.74%

## Code

```typescript
function shuffle(nums: number[], n: number): number[] {
  const res: number[] = [];
  for (let i = 0; i < n; i++) res.push(nums[i], nums[i + n]);
  return res;
}
```
