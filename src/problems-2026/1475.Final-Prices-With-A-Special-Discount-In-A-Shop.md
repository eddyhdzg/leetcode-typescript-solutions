# 1475. Final Prices With A Special Discount In A Shop

### O(n) - TS with Stack

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

Runtime: 2 ms · Beats 33.81%
Memory: 58.72 MB · Beats 32.66%

## Code

```typescript
function finalPrices(prices: number[]): number[] {
  const stack: number[] = [];
  const answer: number[] = new Array(prices.length);

  for (let i = prices.length - 1; i >= 0; i--) {
    const price = prices[i];

    while (stack.length && price < stack.at(-1)!) {
      stack.pop();
    }

    if (stack.length && price >= stack.at(-1)!) {
      answer[i] = price - stack.at(-1)!;
    } else {
      answer[i] = price;
    }

    stack.push(price);
  }

  return answer;
}
```
