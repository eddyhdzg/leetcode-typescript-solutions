# 1512. Number Of Good Pairs

### O(n) - TS with hash map

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

Memory:  · Beats

## Code

```typescript
function numIdenticalPairs(nums: number[]): number {
  const count: Record<number, number> = {};
  let res = 0;

  // Count occurrences: [1,2,3,1,1,3] → {1:3, 2:1, 3:2}
  nums.forEach((num) => {
    count[num] = (count[num] ?? 0) + 1;
  });

  // Each time we see num again, it pairs with every prior one not yet counted.
  // 1s: 2+1=3 pairs, 3s: 1 pair → res = 4
  nums.forEach((num) => {
    if (count[num] > 1) {
      res += --count[num];
    }
  });

  return res;
}
```
