# 1855. Maximum Distance Between A Pair Of Values

### O(n + m) - TS with two pointers

## Complexity

- Time: $$O(n + m)$$
- Space: $$O(1)$$

## Submission

Runtime: 2 ms · Beats 86.67%
Memory: 68.40 MB · Beats 66.67%

## Code

```typescript
function maxDistance(nums1: number[], nums2: number[]): number {
  let i = 0;
  let j = 0;
  let res = 0;

  while (i < nums1.length && j < nums2.length) {
    if (nums1[i] <= nums2[j]) {
      res = Math.max(res, j - i);
      j++;
    } else {
      i++;
    }
  }

  return res;
}
```
