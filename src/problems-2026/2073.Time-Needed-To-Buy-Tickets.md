# 2073. Time Needed To Buy Tickets

### O(n) - TS with single pass

## Complexity

- Time: $$O(n)$$
- Space: $$O(1)$$

## Submission

## Code

```typescript
function timeRequiredToBuy(tickets: number[], k: number): number {
  let res = 0;

  tickets.forEach((ticket, i) => {
    if (i <= k) res += Math.min(tickets[k], ticket);
    else res += Math.min(tickets[k] - 1, ticket);
  });

  return res;
}
```
