# 2833. Furthest Point From Origin

### O(n) - TS with counting

## Complexity

- Time: $$O(n)$$
- Space: $$O(1)$$

## Submission

Runtime: 2 ms · Beats 54.55%
Memory: 56.52 MB · Beats 100.00%

## Code

```typescript
function furthestDistanceFromOrigin(moves: string): number {
  let l = 0, r = 0, u = 0;

  for (const ch of moves) {
    if (ch === "L") l++;
    if (ch === "R") r++;
    if (ch === "_") u++;
  }

  return Math.abs(l - r) + u;
}
```
