# 485. Max Consecutive Ones

### O(n) - TS with sliding window

## Complexity

- Time: $$O(n)$$
- Space: $$O(1)$$

## Submission

Runtime: 2 ms · Beats 65.97%
Memory: 60.38 MB · Beats 81.8%

## Code

```typescript
function findMaxConsecutiveOnes(nums: number[]): number {
  let aux = 0;
  let res = 0;

  for (let i = 0; i < nums.length; i++) {
    if (nums[i]) {
      aux++;
      res = Math.max(res, aux);
    } else {
      aux = 0;
    }
  }

  return res;
}
```
