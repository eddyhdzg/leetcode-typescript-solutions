# 61. Rotate List

### O(n) - TS with two pointers

## Complexity

- Time: $$O(n)$$
- Space: $$O(1)$$

## Submission

## Code

```typescript
function rotateRight(head: ListNode | null, k: number): ListNode | null {
  if (k === 0 || !head || !head.next) return head;

  let len = 1;
  let tail = head;
  while (tail.next) {
    tail = tail.next;
    len++;
  }

  k %= len;
  if (k === 0) return head;

  let newTail = head;
  for (let i = 0; i < len - k - 1; i++) {
    newTail = newTail.next!;
  }

  const newHead = newTail.next;
  newTail.next = null;
  tail.next = head;
  return newHead;
}
```
