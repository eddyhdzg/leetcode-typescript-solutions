# 645. Set Mismatch

### O(n) - TS with Set()

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

Runtime: 11 ms · Beats 59.53%
Memory: 63.56 MB · Beats 40.58%

## Code

```typescript
function findErrorNums(nums: number[]): number[] {
  let twice = 0;
  let missing = 0;
  const set = new Set<number>();

  for (const num of nums) {
    if (set.has(num)) twice = num;
    else set.add(num);
  }
  for (let i = 1; !missing && i <= nums.length; i++) {
    if (!set.has(i)) {
      missing = i;
      break;
    }
  }

  return [twice, missing];
}
```
