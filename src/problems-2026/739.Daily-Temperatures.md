# 739. Daily Temperatures

### O(n) - TS with monotonic stack

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

Runtime: 41 ms · Beats 21.95%
Memory: 80.89 MB · Beats 24.43%

## Code

```typescript
interface Day {
  idx: number;
  temp: number;
}

function dailyTemperatures(temperatures: number[]): number[] {
  const stack: Day[] = [];
  const answer = new Array(temperatures.length).fill(0);

  for (let i = 0; i < temperatures.length; i++) {
    const temp = temperatures[i];

    while (stack.length && temp > stack.at(-1)!.temp) {
      answer[stack.at(-1)!.idx] = i - stack.at(-1)!.idx;
      stack.pop();
    }

    stack.push({ temp, idx: i });
  }

  return answer;
}
```
