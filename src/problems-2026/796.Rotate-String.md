# 796. Rotate String

### O(n) - TS with string doubling

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

Runtime: 0 ms · Beats 100.00%
Memory: 55.62 MB · Beats 31.50%

## Code

```typescript
function rotateString(s: string, goal: string): boolean {
  return s.length === goal.length && (s + s).includes(goal);
}
```
